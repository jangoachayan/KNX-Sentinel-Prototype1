# KNX Sentinel (Prototype 1)

**Advanced KNX Diagnostics and Anomaly Detection Agent for Home Assistant.**

KNX Sentinel is a Home Assistant Add-on designed to bring "Edge Intelligence" to your KNX installation. It monitors bus traffic, detects statistical anomalies in sensor data, and pushes diagnostics to external systems (InfluxDB/MQTT) for long-term retention and analysis.

## Installation

This repository is a **Home Assistant Add-on Repository**.

1.  Copy the URL of this repository:
    `https://github.com/jangoachayan/KNX-Sentinel-Prototype1`
2.  In Home Assistant, go to **Settings > Add-ons > Add-on Store**.
3.  Click the **three dots** > **Repositories**.
4.  Paste the URL and click **Add**.
5.  Install **KNX Sentinel**.
